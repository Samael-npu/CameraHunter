\## 📌 产品概述

Perception Radar 是一款集成化 TSCM (技术侦察反制) 终端。系统通过调度移动设备硬件传感器阵列，结合多模态融合算法，针对环境中的隐蔽监听、监控设备进行深度审计与物理标定。



\## 🛡️ 核心技术特性

\* \*\*多模态融合判定 (Multimodal Fusion)\*\*：实时联动磁场通量、高频声电能量及射频(RF)信号强度，通过动态权重的专家模型计算环境风险置信度。

\* \*\*差分惯性寻向 (Spatial Differential)\*\*：利用伪合成孔径(Pseudo-SAR)算法，在用户移动过程中补偿背景噪声，实现对辐射源物理方位的精准锁定。

\* \*\*协议层深度扫描 (Protocol Audit)\*\*：涵盖 SSDP/UPnP 组播嗅探、RTSP 协议指纹识别及常用安防端口探测，揭示局域网内隐藏的物联网节点。

\* \*\*ICMP 波动指纹分析 (Thermal Analysis)\*\*：通过对目标节点执行高频探针注入，分析 RTT 方差变化，利用热节流特征识别未采取散热措施的微型窃密芯片。



\## 📖 快速入门

1\. \*\*环境校准\*\*：启动后保持设备静止2秒，系统将自动采集环境磁场与声电基准。

2\. \*\*感知探测\*\*：手持设备在室内缓慢移动，关注磁通量及射频信号的动态曲线。

3\. \*\*物理标定\*\*：当发现高置信度目标时，切换至【罗盘】页面，根据差分红点指示的方向进行排查。
