Perception Radar 3.0 是一款基于**物理特性**而非单纯名单匹配的安全工具。它通过手机内置的精密磁传感器与射频模块，探测隐藏在环境中的偷拍设备、及未经授权的网络节点。

## 🌟 核心功能

### 1. 核心源探测 (Focus Mode)
不同于普通的金属探测，利用数学模型分析磁场能量的**聚焦度**。能有效分辨平缓的背景金属（如钢筋）与极度聚焦的电子元件核心。

### 2. 物理锁定 (Sync Locking)
当环境中存在射频信号（蓝牙/Wi-Fi）且其波动与磁场变化高度**同步**时，雷达会自动锁定该目标为“物理关联设备”，判定其正在实时采集并外传数据。

### 3. 2D 寻迹
- **方位解算**：实时指向潜在威胁源。
- **动态测距**：基于路径损耗模型估算设备距离。
- **环境自适应**：自动识别物理遮挡（如穿墙）并修正测距精度。

### 4. 内核 (Deep Scan)
利用 Root 权限（非必须，建议使用）绕过安卓协议栈限制，执行 **JitterPrint™ 协议栈测谎**，识别那些修改了 MAC 地址的伪装设备。

## 📖 使用说明

### 第一步：环境初始化
1. **获取权限**：启动 App 后请授予“精确地理位置”与“蓝牙扫描”权限（Android 12+ 必备）。
2. **基准校准**：在远离大功率电器的开阔地带，点击“启动感知”，App 会自动解算当前环境的**底噪基准**。

### 第二步：感知与扫描
1. **主页面 (感知)**：
   - 观察“物理能量流”图表。如果波形突然变窄且数值飙升，说明附近有活跃电子核心。
   - 查看“射频感应节点”。**同步率 (Sync Score) 超过 75%** 的节点会变红，需优先排查。
2. **底层审计**：滑动到底部，查看网络审计节点。识别异常 IP 或具有高危端口开放的局域网设备。

### 第三步：方位寻迹 (Tactical Compass)
1. 切换到“寻迹”标签页。
2. **红点**代表已被系统物理锁定的高风险设备。
3. **关键操作**：手持手机围绕房间中心缓慢走动。通过这种“多点定位”操作，雷达能通过运动方差解算出更精确的物理坐标。

### 第四步：控制与进阶设置
1. 在“控制”页面，可以调节**锁定灵敏度**。
2. 如果设备已 Root，建议开启“内核测谎”以获得更高精度的指纹识别能力。

## 🛠️ 技术指标
- **磁感应采样**：100Hz 高频捕获。
- **测距模型**：MC-APL (Motion Coupled Adaptive Path Loss)。
- **平滑引擎**：基于卡尔曼阻尼的 SignalProcessor。
